# メモリーチャイム ～鐘を打て～（GameJam_2025）

## 概要
本作品は、ゲームジャムで制作した記憶ゲームです。  
コントローラーを使用し、赤く点灯した鐘の位置を記憶して順番通りに叩いていくゲームとなっています。

ゲームが進行するにつれて記憶量が増加し、難易度が段階的に上昇します。  
また、スコアの保存・比較が可能なランキング機能を実装しており、繰り返しプレイによる記録更新を楽しめる設計となっています。

## 制作背景
ゲームジャムのテーマ「うつ」をもとに企画しました。  
「鐘を打つ」という発想から、音と操作を組み合わせた記憶ゲームとして設計しています。

シンプルなルールながらも、記憶力と集中力を求めることで、短時間でも繰り返し遊べるゲームを目指しました。

## 工夫した点

### ■ タイトル画面の操作処理
- カーソル移動処理を実装し、入力に応じた選択切り替えを実現
- 選択状態が視覚的に分かるようにUIを調整

### ■ ランキング処理
- スコアの比較・並び替え処理を実装
- 上位スコアが正しく表示されるよう設計
- プレイヤーが結果を把握しやすい表示を意識

## 苦労した点
- 短期間での開発における仕様調整と実装の両立
- チーム内での認識の共有・すり合わせ
